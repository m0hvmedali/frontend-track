01 مقدمة CSS والمحددات

ما هو CSS؟

CSS (Cascading Style Sheets) هي لغة تصميم تستخدم لتحديد شكل ومظهر صفحات الويب. تعمل مع HTML لإنشاء مواقع جميلة وتفاعلية.

مثال: CSS الأساسي
<div class="welcome-box">
    <h1>مرحباً بك في عالم CSS!</h1>
    <p>هذا مثال بسيط على قوة CSS</p>
    <button class="btn">ابدأ التعلم</button>
</div>
.welcome-box {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
    padding: 2rem;
    border-radius: 15px;
    text-align: center;
    box-shadow: 0 10px 30px rgba(0,0,0,0.3);
}

.welcome-box h1 {
    font-size: 2.5rem;
    margin-bottom: 1rem;
    animation: fadeInUp 1s ease;
}

.welcome-box p {
    font-size: 1.2rem;
    margin-bottom: 2rem;
    opacity: 0.9;
}

.btn {
    background: #ff6b6b;
    color: white;
    padding: 12px 30px;
    border: none;
    border-radius: 25px;
    font-size: 1.1rem;
    cursor: pointer;
    transition: all 0.3s ease;
}

.btn:hover {
    background: #ff5252;
    transform: translateY(-2px);
    box-shadow: 0 5px 15px rgba(255,107,107,0.4);
}

@keyframes fadeInUp {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

المحددات (Selectors)

المحددات تخبر CSS أي عناصر HTML يجب تطبيق الأنماط عليها. هناك أنواع مختلفة من المحددات لاستهداف العناصر بطرق متنوعة.

مثال: أنواع المحددات
<div class="container">
    <h1>عنوان رئيسي</h1>
    <h2 class="subtitle">عنوان فرعي</h2>
    <p id="intro">هذه فقرة مقدمة مهمة</p>
    <p>فقرة عادية</p>
    <ul>
        <li>العنصر الأول</li>
        <li class="special">العنصر المميز</li>
        <li>العنصر الثالث</li>
    </ul>
    <button>زر عادي</button>
</div>
/* محدد العنصر */
h1 {
    color: #2c3e50;
    font-size: 2.5rem;
    text-align: center;
}

/* محدد الفئة */
.subtitle {
    color: #3498db;
    font-size: 1.8rem;
    border-bottom: 2px solid #3498db;
    padding-bottom: 0.5rem;
}

/* محدد المعرف */
#intro {
    background: #f8f9fa;
    padding: 1rem;
    border-left: 4px solid #e74c3c;
    font-weight: bold;
}

/* محدد الطفل */
ul li {
    padding: 0.5rem;
    margin: 0.25rem 0;
    background: #ecf0f1;
    border-radius: 5px;
}

/* محدد الفئة المتداخل */
li.special {
    background: #f39c12;
    color: white;
    font-weight: bold;
}

/* محدد الحالة */
button:hover {
    background: #27ae60;
    color: white;
    cursor: pointer;
    transform: scale(1.05);
    transition: all 0.3s ease;
}

/* محدد الطفل المباشر */
.container > p {
    line-height: 1.6;
    margin: 1rem 0;
}

02 الألوان والخطوط والنصوص

الألوان والخلفيات

الألوان هي أساس التصميم الجذاب. CSS توفر طرق متعددة لتحديد الألوان والخلفيات المتقدمة.

مثال: الألوان والتدرجات
<div class="color-showcase">
    <div class="color-box solid">لون صلب</div>
    <div class="color-box gradient">تدرج خطي</div>
    <div class="color-box radial">تدرج دائري</div>
    <div class="color-box pattern">نمط متكرر</div>
</div>
.color-showcase {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 1rem;
    padding: 2rem;
}

.color-box {
    height: 150px;
    display: flex;
    align-items: center;
    justify-content: center;
    color: white;
    font-weight: bold;
    font-size: 1.2rem;
    border-radius: 10px;
    transition: transform 0.3s ease;
}

.color-box:hover {
    transform: translateY(-5px);
}

.solid {
    background-color: #e74c3c;
}

.gradient {
    background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
}

.radial {
    background: radial-gradient(circle, #667eea, #764ba2);
}

.pattern {
    background: 
        linear-gradient(45deg, #ff9a9e 25%, transparent 25%),
        linear-gradient(-45deg, #ff9a9e 25%, transparent 25%),
        linear-gradient(45deg, transparent 75%, #fecfef 75%),
        linear-gradient(-45deg, transparent 75%, #fecfef 75%);
    background-size: 20px 20px;
    background-position: 0 0, 0 10px, 10px -10px, -10px 0px;
}

الخطوط والنصوص

الخطوط والنصوص هي وسيلة التواصل الأساسية في الويب. CSS توفر تحكماً كاملاً في مظهر النصوص.

مثال: تنسيق النصوص
<div class="typography-demo">
    <h1 class="main-title">عنوان رئيسي أنيق</h1>
    <h2 class="sub-title">عنوان فرعي مميز</h2>
    <p class="intro-text">
        هذا نص مقدمة يوضح كيفية تنسيق النصوص بطريقة جميلة ومقروءة.
        يمكن استخدام خصائص مختلفة لتحسين مظهر النص.
    </p>
    <p class="highlight-text">نص مميز بتأثيرات خاصة</p>
    <blockquote class="quote">
        "التصميم ليس فقط كيف يبدو الشيء، بل كيف يعمل أيضاً"
        <cite>- ستيف جوبز</cite>
    </blockquote>
</div>
.typography-demo {
    max-width: 800px;
    margin: 0 auto;
    padding: 2rem;
    font-family: 'Cairo', sans-serif;
}

.main-title {
    font-size: 3rem;
    font-weight: 900;
    background: linear-gradient(45deg, #667eea, #764ba2);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
    text-align: center;
    margin-bottom: 1rem;
    text-shadow: 2px 2px 4px rgba(0,0,0,0.1);
}

.sub-title {
    font-size: 1.8rem;
    color: #2c3e50;
    text-align: center;
    margin-bottom: 2rem;
    position: relative;
}

.sub-title::after {
    content: '';
    position: absolute;
    bottom: -10px;
    left: 50%;
    transform: translateX(-50%);
    width: 100px;
    height: 3px;
    background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
    border-radius: 2px;
}

.intro-text {
    font-size: 1.1rem;
    line-height: 1.8;
    color: #34495e;
    text-align: justify;
    margin-bottom: 2rem;
}

.highlight-text {
    font-size: 1.3rem;
    font-weight: bold;
    color: #e74c3c;
    text-transform: uppercase;
    letter-spacing: 2px;
    text-align: center;
    margin: 2rem 0;
    animation: pulse 2s infinite;
}

.quote {
    background: #f8f9fa;
    border-left: 5px solid #3498db;
    padding: 1.5rem;
    margin: 2rem 0;
    font-style: italic;
    font-size: 1.2rem;
    border-radius: 0 10px 10px 0;
}

.quote cite {
    display: block;
    text-align: right;
    margin-top: 1rem;
    font-weight: bold;
    color: #7f8c8d;
}

@keyframes pulse {
    0%, 100% { opacity: 1; }
    50% { opacity: 0.7; }
}
